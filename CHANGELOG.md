## [0.4.0] - 2025-01-09

### Added
- Smart PDF splitting with multiple strategies (fixed, hybrid, enhanced)
- Batch processing with parallel execution support
- Document reassembly and validation
- CLI interface with analyze, chunk, convert, and validate commands
- Python API for programmatic usage
- Comprehensive test suite with integration tests
- CI/CD pipeline with GitHub Actions
- Pre-commit hooks for code quality
- Security scanning with pip-audit
- Code quality tools: ruff, mypy, pytest
- Developer documentation and contributing guidelines
- Code of Conduct (Contributor Covenant 3.0)
- Issue and PR templates
- Dependabot for automated dependency updates
